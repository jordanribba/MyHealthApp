# My Health App

Aplicación Flutter que incluye:
- Calculadora de edad
- Calculadora de IMC
- Determinador de signo zodiacal

Desarrollada con una estructura basada en Clean Architecture.

## Cómo ejecutar la app

### Requisitos
- Flutter SDK instalado
- Emulador o dispositivo móvil conectado

### Pasos
1. Clona el repositorio:
   ```bash
   git clone https://github.com/jordanribba/MyHealthApp.git


2. entrar al proyecto:
   cd my-healt-app

3. Instala dependencias
   flutter pub get
   
5. Ejecuta la app
   flutter run
